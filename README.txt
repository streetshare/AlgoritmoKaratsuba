Pasos para ejecutar el aplicativo Karatsuba

1. Directamente desde el Aplicativo
	a. Descargar el archivo "Karatsuba.exe"
	b. Seguir las instrucciones del aplicativo para la operacion de los numeros.
	
2. Desde la interfaz de Visual Studio
	a. Descargar el archivo "Karatsuba.rar"
	b. Descomprimir el archivo anterior en una carpeta local
	c. Cuando se descomprime el archivo, se encontrará una carpeta llamada
		"Karatsuba"
	d. Iniciar el Software de Visual Studio
	e. En la Pagina Principal, seleccionar la opcion de "Abrir Proyecto" y buscar
		dentro de la carpeta "Karatsuba" el archivo "Karatsuba.vbproj"
	f. Una vez abierto el proyecto, se podrá editar o simplemente ejecutar con la
		tecla "F5" o en la parte superior el icono verde que dice "Iniciar"
